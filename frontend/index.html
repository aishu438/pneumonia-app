Architecture

Convolutional Neural Network

The model uses a CNN with multiple convolutional and pooling layers for hierarchical feature extraction, culminating in a softmax binary classifier. Trained with the Adam optimizer and sparse categorical crossentropy loss.

Preprocessing

Standardized Input Pipeline

Each chest X-ray is resized to 180×180 pixels and normalized to a [0, 1] pixel value scale for computational stability. The model detects edges, shapes, and opacities through progressive feature learning.

Dataset

ChestXRay Kaggle Dataset

Trained on over 5,000 labeled chest radiographs from pediatric patients. Images are classified as Normal or Pneumonia (bacterial/viral). The model identifies fluid accumulation in the alveoli appearing as white patches.
